## Bubble データタイプ一覧

Bubble では「Data Types」としてデータを管理します。各データタイプのフィールドを定義します。

## データタイプの関連図

```
User
 └── has many → （関連データタイプ）
      └── belongs to → User

（データタイプA）
 └── has many → （データタイプB）
```

---

## Bubble のフィールドタイプ一覧

| タイプ | 説明 |
|-------|------|
| text | テキスト文字列 |
| number | 数値 |
| date | 日時 |
| yes/no | Boolean値 |
| image | 画像ファイル |
| file | ファイル |
| geographic address | 住所・位置情報 |
| User | User データタイプへの参照 |
| （カスタムタイプ） | 自作データタイプへの参照 |
| list of（タイプ） | 複数値のリスト |
