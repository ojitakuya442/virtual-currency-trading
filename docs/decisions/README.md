# decisions/ — 司令塔の判断ログ

司令塔（AIセッション）が行った判断・変更をすべてここに記録する。書式は `docs/opus-command-center-spec.md` §9 を参照。

| ファイル名 | 内容 |
|-----------|------|
| `YYYY-MM-DD-weekly.md` | 週次パフォーマンスレビュー（§9.1） |
| `YYYY-MM-DD-tune-<bot名>.md` | パラメータ変更の記録（§9.2）。評価予定日と成功基準を必ず含める |
| `YYYY-MM-DD-incident-<概要>.md` | インシデント対応の記録 |

**ルール**: 記録のない変更はロールバック対象とみなす。過去の判断を確認せずに同種の変更を繰り返さないこと。
